# 📌 Investment Calculator 💰
A simple React app to calculate your investments to the interest and the years given.

## 🚀 Features
- Filter investment by year, interest, so much more
- Responsive UI with CSS Flexbox

## 🔧 Installation
1. Clone the repo:
git clone 'project url link'
or 
📚Download the zip file and extract it 
cd 'file directory' 
npm install

3. Run the app:
▶️'npm start' or 'npm run dev' to start it
